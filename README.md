
# Employee Management System

A Java web application built using Apache Tapestry, Spring, and Hibernate.

## Features

- User Login Authentication
- Login Validation using Service Layer
- Error Message for Invalid Login
- Employee List Page using Tapestry Grid
- View Individual Employee Details
- Add New Employee
- Form Validation
- Reusable Header Component for Company Info
- Spring + Hibernate Integration
- DAO Layer using Criteria API

## Tech Stack

- Java
- Apache Tapestry
- Spring Framework
- Hibernate ORM
- Maven
- MySQL / Oracle DB

## Architecture

Presentation Layer -> Tapestry Pages  
Service Layer -> Spring Services  
DAO Layer -> Hibernate DAO

## Pages

- Login Page
- Employee List Page
- Employee Details Page
- Add Employee Page

## How to Run

1. Clone project
2. Configure DB in properties file
3. Run Maven build
4. Deploy on Tomcat / Jetty
